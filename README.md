# 🐍 Python + AI 辅助编程课程学习网站

> 专为学生打造的交互式编程学习平台，整合Python核心语法、Flask框架、AI辅助编程等完整知识点，提供沉浸式学习体验。

## ✨ 功能特性

### 🎯 完整知识体系
- **Python核心语法**：变量、数据类型、条件判断、循环、函数、列表、字典等基础知识点
- **Flask Web框架**：应用创建、路由、模板引擎、模板继承等Web开发技能
- **AI辅助编程**：四步方法论（角色初始化、代码追踪、模块适配、迭代调试）
- **前端技术**：HTML基础、CSS选择器、Flexbox布局
- **版本控制**：Git基础命令、GitHub/Gitee Pages部署
- **计算思维**：问题分解、模式识别、抽象、算法设计

### 🎨 现代化UI设计
- **渐变背景**：紫色渐变主题，视觉效果优雅
- **毛玻璃效果**：半透明卡片+背景模糊，科技感十足
- **双主题支持**：🌞 明亮模式 / 🌙 暗色模式自由切换，自动记忆偏好
- **响应式布局**：完美适配桌面端、平板、手机等各种设备
- **平滑动画**：悬停效果、过渡动画、滚动效果，提升交互体验

### 📚 学习友好功能
- **侧边导航栏**：完整目录结构，点击快速跳转对应章节
- **导航高亮**：滚动时自动高亮当前阅读章节，随时知道学习位置
- **折叠面板**：知识点分类折叠，按需查看，减少信息干扰
- **代码高亮**：所有代码块深色主题高亮，语法清晰易读
- **知识点卡片**：AI四步方法论卡片化展示，直观易懂
- **返回顶部**：快速回到页面顶部，方便长文档阅读
- **平滑滚动**：所有锚点跳转平滑过渡，不突兀

### 📊 可视化提示
- **表格美化**：渐变表头，悬停效果，数据清晰展示
- **提示框**：💡 普通提示、⚠️ 重要提示分类展示
- **标签样式**：行内代码标签化，突出显示

## 🚀 使用方法

### 本地使用
1. 直接下载项目中的 `course.html` 文件
2. 用 Chrome、Edge、Firefox等现代浏览器打开即可
3. 无需安装任何依赖，无需网络环境也能正常使用

### 功能操作
- **切换主题**：点击右上角 🌙/☀️ 按钮切换明暗主题
- **导航跳转**：点击左侧目录快速跳转到对应章节
- **折叠展开**：点击折叠面板标题展开/收起内容
- **返回顶部**：滚动页面后点击右下角 ↑ 按钮回到顶部

## 📦 部署指南

### 部署到 GitHub Pages
1. Fork 本项目到你的 GitHub 账户
2. 进入仓库 Settings → Pages
3. Source 选择 main 分支，根目录
4. 保存后等待部署完成，访问 `https://你的用户名.github.io/仓库名` 即可

### 部署到 Gitee Pages
1. 将代码推送到 Gitee 仓库
2. 进入仓库 → 服务 → Gitee Pages
3. 选择部署分支，启动服务
4. 访问 `https://你的用户名.gitee.io/仓库名` 即可

### 其他部署方式
- 可以部署到任何支持静态网站的托管服务（Vercel、Netlify、云服务器等）
- 直接上传 `course.html` 文件到服务器根目录即可

## 🎨 自定义配置

### 修改主题色
在 `course.html` 的 CSS 部分找到 `:root` 变量定义：
```css
:root {
    --primary-color: #667eea;    /* 主色调 */
    --secondary-color: #764ba2;  /* 辅助色 */
    /* 其他变量... */
}
```
修改对应颜色值即可更换整体主题配色。

### 添加新知识点
在对应的 `<div class="section">` 标签内添加内容即可，保持现有格式一致：
- 标题使用 `<h2>`、`<h3>`、`<h4>`
- 代码块使用 `<pre>` 标签
- 表格使用 `<table>` 标签
- 提示框使用 `<div class="note">` 或 `<div class="important">`
